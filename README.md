## How to run the app

**Cloning the Repository**

```bash
git clone https://github.com/RKeyProg/simpleCalcInnowise.git
cd iphone15-react
```

**Installation**

Install the project dependencies using npm:

```bash
npm install
```

**Running the Project**

```bash
npm run dev
```

**Building the Project**

```bash
npm run build
```
